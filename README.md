# Arduino SPI EEPROM
Simple project to read and write a ST 95080 8Kbit/1KB EEPROM.

## PINS
### ST 95080
1 - D10<br>
2 - D12<br>
3 - VSS (5v)<br>
4 - GND<br>
5 - D11<br>
6 - D13<br>
7 - VSS (5v)<br>
8 - VSS (5v)<br>

### Write button
Simple push button with ground resistor that gives high signal in D5.
